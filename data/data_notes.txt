Для запуска расчета заполнения по конкретному месторождению необходимо поместить в папку data папку с данными по
этому месторождению.

Пример папки с данными по месторождению Отдельное:

Среди всего набора файлов в вышеуказанной папке в проекте используются следующие:
fond.feather - данные о фонде скважин (характер работы)
sost.feather - данные о состоянии скважин (статус работы)
merop.feather - данные о мероприятиях скважин
sh.feather - данные ШТР суточные скважин

Для просмотра таблиц в бинарном формате .feather можно вопользоваться методом в Pandas read_feather.
Все данные были взяты из БД ГРАДа.
